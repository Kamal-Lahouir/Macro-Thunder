**Imports:** PyQt6.QtWidgets (QDialog, QVBoxLayout, QFormLayout, QDialogButtonBox, QSpinBox, QDoubleSpinBox, QComboBox, QLineEdit, QCheckBox, QPushButton, QHBoxLayout, QWidget, QGroupBox, QLabel), PyQt6.QtCore (Qt), PyQt6.QtGui (QKeyEvent). Import all block types from macro_thunder.models.blocks.

**`_qt_key_to_pynput(event: QKeyEvent) -> str`** (module-level private):
Convert QKeyEvent to pynput key string. Build a dict mapping Qt.Key.Key_Space → "Key.space", Key_Return/"Key.enter", Key_Enter/"Key.enter", Key_Backspace/"Key.backspace", Key_Delete/"Key.delete", Key_Tab/"Key.tab", Key_Escape/"Key.esc", Key_Up/"Key.up", Key_Down/"Key.down", Key_Left/"Key.left", Key_Right/"Key.right", Key_Home/"Key.home", Key_End/"Key.end", Key_PageUp/"Key.page_up", Key_PageDown/"Key.page_down", Key_Insert/"Key.insert", Key_F1–Key_F12/"Key.f1"–"Key.f12", Key_CapsLock/"Key.caps_lock", Key_NumLock/"Key.num_lock", Key_ScrollLock/"Key.scroll_lock", Key_Print/"Key.print_screen", Key_Pause/"Key.pause". If key in named dict, return that. Else if event.text() is non-empty and printable, return event.text().lower(). Fallback: return f"Key.{Qt.Key(key).name.replace('Key_', '').lower()}".

**`KeyCaptureField(QWidget)`**:
- `__init__(self, initial_value: str = "", parent=None)`: self._value = initial_value, self._capturing = False. QHBoxLayout, self._display = QLineEdit(initial_value) setReadOnly(True) setMinimumWidth(160), self._btn = QPushButton("Press a key...") setFixedWidth(110). Connect self._btn.clicked → self._start_capture.
- `value(self) -> str`: return self._value.
- `_start_capture(self)`: self._capturing = True, self._display.setText("… press a key …"), self._btn.setText("Cancel"), disconnect and reconnect btn.clicked → self._cancel_capture, setFocusPolicy(StrongFocus), setFocus().
- `_cancel_capture(self)`: self._capturing = False, restore display text to self._value, restore btn to "Press a key..." → self._start_capture, clearFocus().
- `keyPressEvent(self, event)`: if not capturing, super(). Return if modifier-only keys (Shift/Control/Alt/Meta/AltGr). If Escape, call _cancel_capture. Otherwise: self._value = _qt_key_to_pynput(event), update display, reset capturing=False, restore btn, clearFocus(), event.accept().

**`_find_click_partner(block: MouseClickBlock, doc_blocks: list) -> tuple[int, object]`**:
Use doc_blocks.index(block) to find idx. target_direction = "up" if block.direction == "down" else "down". If block.direction == "down", scan forward from idx+1; else scan backward from idx-1. Match: isinstance(b, MouseClickBlock) and b.button == block.button and b.direction == target_direction. Return (i, b) on first match. Return (-1, None) if not found. Wrap doc_blocks.index in try/except ValueError → return (-1, None).

**Per-type dialog classes** (all follow Pattern 2: write to block only in accept()):

`MouseClickEditDialog(QDialog)`: fields x (QSpinBox -9999..9999), y (QSpinBox -9999..9999), button (QComboBox: "left","right","middle"), direction (QComboBox: "down","up","click"). If partner is not None, show "(editing paired down/up blocks)" as a QLabel under the form. In accept(): apply x, y, button, direction to self._block; apply x, y, button to self._partner (if not None; partner direction stays unchanged).

`MouseMoveEditDialog(QDialog)`: fields x, y (QSpinBox -9999..9999 each). accept(): apply to block.

`MouseScrollEditDialog(QDialog)`: fields x, y (QSpinBox -9999..9999), dx, dy (QSpinBox -99..99). accept(): apply.

`KeyPressEditDialog(QDialog)`: KeyCaptureField for key, QComboBox direction ("down","up","key"). accept(): block.key = self._key_field.value(), block.direction = self._dir_combo.currentText().

`DelayEditDialog(QDialog)`: QDoubleSpinBox duration range 0.001..3600.0 decimals=3 suffix=" s". accept(): block.duration = value.

`LabelEditDialog(QDialog)`: QLineEdit name with placeholder "Unique label name". accept(): block.name = text.strip().

`GotoEditDialog(QDialog)`: QLineEdit target with placeholder "Label name to jump to". accept(): block.target = text.strip().

`WindowFocusEditDialog(QDialog)`: All fields from WindowFocusBlock. Use QFormLayout. Fields: executable (QLineEdit), title (QLineEdit), match_mode (QComboBox: "Contains","Exact","Starts With"), timeout (QDoubleSpinBox 0..3600 decimals=1 suffix=" s"), on_failure_label (QLineEdit), on_success_label (QLineEdit). Then a QCheckBox "Reposition window" (self._reposition_check). Below that, a QGroupBox "Position/Size" (self._reposition_group) with x, y, w, h QSpinBoxes (range -9999..9999). Connect self._reposition_check.toggled → self._reposition_group.setVisible. Set initial visibility from block.reposition. accept(): write all fields to block.

`LoopStartEditDialog(QDialog)`: QSpinBox repeat range 1..9999. accept(): block.repeat = value.

**`open_edit_dialog(block, doc_blocks: list, parent=None) -> bool`** (public):
Dispatcher. isinstance chain: MouseClickBlock → _find_click_partner → MouseClickEditDialog(block, partner, parent). MouseMoveBlock → MouseMoveEditDialog. MouseScrollBlock → MouseScrollEditDialog. All dialog classes: setWindowTitle("Edit {BlockTypeName}"), setMinimumWidth(320), use QVBoxLayout with QFormLayout then QDialogButtonBox(Ok|Cancel), buttons.accepted → self.accept, buttons.rejected → self.reject.
