    **src/macro_thunder/settings.py** — AppSettings dataclass:

    ```python
    from __future__ import annotations
    import dataclasses, json, pathlib

    SETTINGS_DIR = pathlib.Path.home() / "Documents" / "MacroThunder"
    SETTINGS_FILE = SETTINGS_DIR / "settings.json"

    @dataclasses.dataclass
    class AppSettings:
        hotkey_start_record: str = "<f9>"
        hotkey_stop_record: str = "<f10>"
        hotkey_start_play: str = "<f6>"
        hotkey_stop_play: str = "<f8>"
        mouse_threshold_px: int = 3

        @classmethod
        def load(cls) -> "AppSettings":
            if SETTINGS_FILE.exists():
                data = json.loads(SETTINGS_FILE.read_text(encoding="utf-8"))
                fields = {f.name for f in dataclasses.fields(cls)}
                return cls(**{k: v for k, v in data.items() if k in fields})
            return cls()

        def save(self) -> None:
            SETTINGS_DIR.mkdir(parents=True, exist_ok=True)
            SETTINGS_FILE.write_text(
                json.dumps(dataclasses.asdict(self), indent=2),
                encoding="utf-8"
            )
    ```

    **src/macro_thunder/hotkeys.py** — HotkeyManager:

    HotkeyManager(QObject parent) extends QObject:
    - Defines pyqtSignals: start_record, stop_record, start_play, stop_play (all no-arg)
    - _hotkey_queue: queue.Queue
    - Starts a QTimer(16ms) drain that checks _hotkey_queue and emits the corresponding signal
    - register(settings: AppSettings):
        Stops any existing GlobalHotKeys listener (if running).
        Builds hotkey_map:
          {
            settings.hotkey_start_record: lambda: _hotkey_queue.put("start_record"),
            settings.hotkey_stop_record: lambda: _hotkey_queue.put("stop_record"),
            settings.hotkey_start_play: lambda: _hotkey_queue.put("start_play"),
            settings.hotkey_stop_play: lambda: _hotkey_queue.put("stop_play"),
          }
        Creates keyboard.GlobalHotKeys(hotkey_map); sets .daemon=True; calls .start().
        Stores listener as self._listener.
    - stop():
        If self._listener exists: call self._listener.stop(); set to None.

    CRITICAL: GlobalHotKeys callbacks run on the listener thread — the lambdas only call
    _hotkey_queue.put(). The QTimer drain on the main thread emits signals. Never call Qt methods
    from inside the lambda (project CLAUDE.md threading rule).

    drain() slot (connected to QTimer):
        while not _hotkey_queue.empty():
            action = _hotkey_queue.get_nowait()
            if action == "start_record": emit self.start_record
            elif action == "stop_record": emit self.stop_record
            elif action == "start_play": emit self.start_play
            elif action == "stop_play": emit self.stop_play
  </action>
