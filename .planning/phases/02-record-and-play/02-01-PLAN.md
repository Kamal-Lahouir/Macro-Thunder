  <name>RecorderService</name>
  <files>src/macro_thunder/recorder/__init__.py, tests/test_recorder.py</files>
  <behavior>
    RecorderService accepts a `queue.Queue` and a pixel threshold (int, default 3).
    It manages two pynput Listeners (mouse + keyboard).

    start(record_start_time: float | None = None):
      - Sets internal _record_start = record_start_time or time.perf_counter()
      - Resets _last_move_x, _last_move_y to None
      - Creates and starts mouse.Listener(on_move, on_click, on_scroll, win32_event_filter=_mouse_filter)
      - Creates and starts keyboard.Listener(on_press, on_release, win32_event_filter=_kb_filter)

    stop():
      - Stops both listeners (from external thread — never from inside callback)
      - Safe to call multiple times

    Internal callbacks (all queue events as ActionBlock instances, NEVER touch Qt):
      on_move(x, y):
        - If _last_move_x is None: set last pos, put MouseMoveBlock(x=x, y=y, timestamp=ts)
        - Else: compute math.hypot(x - last_x, y - last_y); skip if < threshold
        - Update _last_move_x, _last_move_y; put MouseMoveBlock(x=x, y=y, timestamp=ts)
      on_click(x, y, button, pressed):
        - put MouseClickBlock(x=x, y=y, button=button.name, direction="down" if pressed else "up", timestamp=ts)
      on_scroll(x, y, dx, dy):
        - put MouseScrollBlock(x=x, y=y, dx=dx, dy=dy, timestamp=ts)
      on_press(key):
        - put KeyPressBlock(key=_key_to_str(key), direction="down", timestamp=ts)
      on_release(key):
        - put KeyPressBlock(key=_key_to_str(key), direction="up", timestamp=ts)

    _key_to_str(key):
      - isinstance(key, keyboard.Key) → f"Key.{key.name}"
      - hasattr(key, 'char') and key.char → key.char
      - else → str(key)

    _mouse_filter(msg, data):
      LLMHF_INJECTED = 0x00000001
      return not bool(data.flags & LLMHF_INJECTED)

    _kb_filter(msg, data):
      LLKHF_INJECTED = 0x00000010  # bit 4 — keyboard struct different from mouse
      return not bool(data.flags & LLKHF_INJECTED)

    Test cases (input → expected output):
      - on_move(100, 200) with threshold=5, no prior move → MouseMoveBlock(x=100, y=200)
      - on_move(103, 202) with threshold=5, last=(100,200) → discarded (hypot=sqrt(13)≈3.6 < 5)
      - on_move(106, 206) with threshold=5, last=(100,200) → MouseMoveBlock(x=106, y=206)
      - on_click(50, 60, mouse.Button.left, True) → MouseClickBlock(button="left", direction="down")
      - on_click(50, 60, mouse.Button.right, False) → MouseClickBlock(button="right", direction="up")
      - on_scroll(10, 20, 0, -3) → MouseScrollBlock(dx=0, dy=-3)
      - on_press(keyboard.Key.shift) → KeyPressBlock(key="Key.shift", direction="down")
      - on_press(keyboard.KeyCode.from_char('a')) → KeyPressBlock(key="a", direction="down")
      - on_release(keyboard.Key.f8) → KeyPressBlock(key="Key.f8", direction="up")
      - _mouse_filter with data.flags=0x1 → False (filtered)
      - _mouse_filter with data.flags=0x0 → True (passes)
      - _kb_filter with data.flags=0x10 → False (filtered)
      - _kb_filter with data.flags=0x00 → True (passes)
  </behavior>
