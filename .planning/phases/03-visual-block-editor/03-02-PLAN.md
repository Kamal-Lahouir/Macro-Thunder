    Column layout (4 columns):
      COL_TYPE = 0      # "MouseMove", "MouseClick", "KeyPress", etc.  OR group label
      COL_VALUE = 1     # main value: coords, key name, duration, etc.
      COL_TIMESTAMP = 2 # timestamp float (or group total duration for group rows)
      COL_EXTRA = 3     # extra info or empty

    _rebuild_display_rows(self):
      Scans self._doc.blocks sequentially. Runs of consecutive MouseMoveBlock (2+ blocks) become a GroupHeaderRow; the existing expanded state is preserved by matching flat_start/flat_end against a dict keyed on flat_start. A lone MouseMoveBlock not adjacent to another becomes a BlockRow. All other block types become BlockRow. If a group is expanded, insert GroupChildRow entries for each block in the run immediately after its GroupHeaderRow.

    data(self, index, role):
      DisplayRole only (other roles return None):
        BlockRow: col 0 → block.type, col 1 → _block_value(block), col 2 → _block_timestamp(block), col 3 → ""
        GroupHeaderRow: col 0 → f"▶ MouseMove Group" or "▼ MouseMove Group" (expanded), col 1 → f"{count} moves → ({last_x}, {last_y})", col 2 → f"{duration:.3f}s", col 3 → f"({first_x}, {first_y})"
        GroupChildRow: col 0 → "  MouseMove" (indented), col 1 → f"({block.x}, {block.y})", col 2 → f"{block.timestamp:.4f}", col 3 → ""

    _block_value(block): Returns human-readable value string:
      MouseMoveBlock → f"({block.x}, {block.y})"
      MouseClickBlock → f"{block.button} {block.direction} @ ({block.x},{block.y})"
      MouseScrollBlock → f"scroll ({block.dx},{block.dy}) @ ({block.x},{block.y})"
      KeyPressBlock → f"{block.key} {block.direction}"
      DelayBlock → f"{block.duration:.3f}s"
      WindowFocusBlock → f"{block.executable} — {block.title}"
      LabelBlock → f":{block.name}"
      GotoBlock → f"→ {block.target}"

    _block_timestamp(block): Returns f"{block.timestamp:.4f}" if block has timestamp attribute else "".

    flags(self, index):
      base = ItemIsEnabled | ItemIsSelectable | ItemIsDragEnabled
      Editable cells:
        GroupHeaderRow + COL_TIMESTAMP → add ItemIsEditable (duration edit)
        GroupHeaderRow + COL_VALUE → add ItemIsEditable (endpoint coords edit — format "x,y")
        GroupHeaderRow + COL_EXTRA → add ItemIsEditable (start coords edit — format "x,y")
        BlockRow: most block types have editable fields — add ItemIsEditable to COL_VALUE and COL_TIMESTAMP for all BlockRow cells (simple approach: always editable for non-group rows)
        GroupChildRow + COL_VALUE → add ItemIsEditable (x,y)
        GroupChildRow + COL_TIMESTAMP → add ItemIsEditable

    toggle_group(self, display_row_index):
      row = self._display_rows[display_row_index]
      assert isinstance(row, GroupHeaderRow)
      self.beginResetModel()
      row.expanded = not row.expanded
      self._rebuild_display_rows()
      self.endResetModel()

    Emit document_modified = pyqtSignal() on any mutation.

    Import guard: from PyQt6.QtCore import QAbstractTableModel, QModelIndex, Qt, pyqtSignal

    setData(self, index, value, role=EditRole):
      GroupHeaderRow + COL_TIMESTAMP: parse float(value), call _rescale_group_duration(self._doc.blocks, row.flat_start, row.flat_end, new_duration), rebuild, emit dataChanged, emit document_modified. Return True.
      GroupHeaderRow + COL_VALUE (end coords, "x,y"): parse two ints, call _rescale_group_coords with same start and new end. Rebuild, emit. Return True.
      GroupHeaderRow + COL_EXTRA (start coords, "x,y"): parse two ints, call _rescale_group_coords with new start and same end. Rebuild, emit. Return True.
      BlockRow / GroupChildRow + COL_VALUE: parse and update the appropriate field on the underlying block (use block.type to determine field). For MouseMoveBlock/MouseClickBlock/MouseScrollBlock: parse "x,y" for coords; for KeyPressBlock: set key; for DelayBlock: set duration=float; for LabelBlock: set name; for GotoBlock: set target. Rebuild, emit. Return True.
      BlockRow / GroupChildRow + COL_TIMESTAMP: parse float(value), set block.timestamp = float(value). Rebuild, emit. Return True.
      Return False for unhandled cases.

    delete_rows(self, display_row_indices: list[int]):
      Collect all flat_indices to delete (expanding groups to all their children if group header selected).
      Sort descending to avoid index shift. Clear selection (emit layoutAboutToBeChanged / layoutChanged around mutation).
      Use beginResetModel/endResetModel pattern:
        (1) compute new blocks list = [b for i,b in enumerate(self._doc.blocks) if i not in flat_to_delete]
        (2) beginResetModel()
        (3) self._doc.blocks[:] = new_blocks
        (4) rebuild (clear expanded state for deleted groups)
        (5) endResetModel()
      Auto-ungroup: _rebuild_display_rows handles naturally — single move becomes BlockRow.
      Emit document_modified.

    insert_block(self, after_display_row: int, block: ActionBlock):
      Determine flat insertion index from the display row (use flat_index for BlockRow/GroupChildRow, flat_end+1 for GroupHeaderRow, 0 if after_display_row == -1).
      beginResetModel()
      self._doc.blocks.insert(flat_insert, block)
      _rebuild_display_rows()
      endResetModel()
      Emit document_modified.

    move_rows_up(self, display_row_indices: list[int]):
      Find the lowest display row index; if it is 0, return. Collect flat blocks to move, swap with block above in flat list, rebuild.
      Use beginResetModel/endResetModel. Emit document_modified.

    move_rows_down(self, display_row_indices: list[int]):
      Symmetric to move_rows_up. If highest display row is last, return.

    Drag-and-drop methods:
      supportedDropActions: return Qt.DropAction.MoveAction
      mimeTypes: return ["application/x-macroblock-rows"]
      mimeData(indexes): Collect unique display row indices (column 0 only). Encode as comma-separated string of flat index ranges "flat_start:flat_end" for groups, "flat_idx" for single rows. Pack into QMimeData.
      dropMimeData(data, action, row, column, parent):
        Always use column = 0 (ignore passed column — prevent column shift bug).
        Decode source flat index list from mimeData.
        Compute destination flat index from drop row (if row == -1 → append to end).
        Move the flat blocks as a unit in self._doc.blocks.
        beginResetModel(), _rebuild_display_rows(), endResetModel().
        Emit document_modified. Return True.

    IMPORTANT: Never mutate self._doc.blocks while iterating it. Always build new list then assign with [:]= slice.
