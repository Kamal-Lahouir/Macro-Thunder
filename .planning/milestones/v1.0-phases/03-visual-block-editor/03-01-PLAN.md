  <name>Rescaling Logic + DisplayRow Types</name>
  <files>src/macro_thunder/models/view_model.py, tests/test_view_model_rescale.py</files>
  <behavior>
    _rescale_group_duration(doc_blocks, flat_start, flat_end, new_duration):
      - Input: list of ActionBlock (all MouseMoveBlock), flat_start/end indices into the list, new_duration float
      - Rescales each block.timestamp so that (last.timestamp - first.timestamp) == new_duration
      - Proportional: block.timestamp = old_start_t + (block.timestamp - old_start_t) * scale
      - scale = new_duration / old_duration; old_duration = blocks[flat_end].timestamp - blocks[flat_start].timestamp
      - No-op if flat_end == flat_start (single move)
      - No-op if old_duration <= 0

    _rescale_group_coords(doc_blocks, flat_start, flat_end, new_start, new_end):
      - Input: blocks, indices, new_start=(x0,y0), new_end=(xN,yN)
      - Linearly interpolates x/y of each intermediate move: t = i / max(n-1, 1)
      - block.x = round(new_start[0] + t * (new_end[0] - new_start[0]))
      - block.y = round(new_start[1] + t * (new_end[1] - new_start[1]))
      - No-op if flat_end == flat_start (single move)

    Test cases:
      duration_rescale: 3 moves at t=[0.0, 0.5, 1.0], new_duration=2.0 → t=[0.0, 1.0, 2.0]
      duration_rescale: single move → unchanged
      duration_rescale: zero old_duration → unchanged
      coord_rescale: 3 moves from (0,0)→(100,100), new_start=(10,20), new_end=(50,60) → intermediate at (30,40)
      coord_rescale: single move → only start/end applied, no crash
  </behavior>
