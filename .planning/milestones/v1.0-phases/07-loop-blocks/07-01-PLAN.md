Engine loop_stack logic (from RESEARCH.md, verified correct):
```python
loop_stack: list[tuple[int, int]] = []  # (loop_start_index, remaining)

if isinstance(block, LoopStartBlock):
    loop_stack.append((i, block.repeat - 1))  # -1: first pass already in progress
    i += 1
    continue

if isinstance(block, LoopEndBlock):
    if loop_stack:
        start_idx, remaining = loop_stack[-1]
        if remaining > 0:
            loop_stack[-1] = (start_idx, remaining - 1)
            i = start_idx + 1
        else:
            loop_stack.pop()
            i += 1
    else:
        i += 1  # orphaned LoopEnd — skip
    continue
```
LoopStart/LoopEnd count as progress (unlike LabelBlock): set `progress_since_last_goto = True` and clear `goto_fire_count` before the isinstance checks.
